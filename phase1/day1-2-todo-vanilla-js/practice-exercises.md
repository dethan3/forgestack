```javascript

    var self = this;
    // ...
}

// 改写为ES6箭头函数：
addTodo = () => {
    // ...
}
```

#### 3.2 变量声明改造
```javascript
// 将所有 var 改为 const/let
// 找出哪些应该用const，哪些用let，为什么？
```

#### 3.3 字符串模板改造
```javascript
// 将这种拼接：
var html = '<div class="' + className + '">' + text + '</div>';

// 改为模板字符串：
const html = `<div class="${className}">${text}</div>`;
```
