# 如何使用本教程

## 环境准备

### 1. 安装 Node.js
确保安装 Node.js 18 或更高版本：

```bash
node --version  # 应该显示 v18.x.x 或更高
```

### 2. 安装依赖

```bash
npm install
```

### 3. 验证安装

```bash
npm test
```

## 学习流程

### 每个章节的学习步骤

```
第 1 步：阅读概念 (15-20 分钟)
    │
    │  打开 chapters/XX-xxx/concepts/ 下的文档
    │  按顺序阅读，理解核心概念
    │
    ▼
第 2 步：动手练习 (30 分钟)
    │
    │  打开 chapters/XX-xxx/exercises/ 下的练习文件
    │  完成 TODO 部分
    │  运行测试验证答案
    │
    ▼
第 3 步：综合挑战 (30-60 分钟)
    │
    │  阅读 challenges/ 下的挑战描述
    │  实现功能（可借助 AI）
    │  运行测试验证
    │  审核代码质量
    │
    ▼
完成，进入下一章节
```

## 运行测试

### 监听模式（推荐）
文件修改后自动重新测试：

```bash
npm run test:watch
```

### 运行特定章节

```bash
npm run test:01  # 第一章
npm run test:02  # 第二章
# ...
```

### 运行所有测试

```bash
npm test
```

## 练习文件格式

练习文件中会有 `TODO` 标记，需要你填写代码：

```javascript
// exercises/01-variables.js

/**
 * 练习 1.1: 变量声明
 * 使用 const 声明一个常量 PI
 */

// TODO: 在下面声明常量
const PI = ______;  // ← 替换 ______ 为正确答案
```

## 测试结果解读

### ✅ 通过
```
 ✓ Exercise 1.1: PI should be 3.14159
```

### ❌ 失败
```
 ✗ Exercise 1.1: PI should be 3.14159
   Expected: 3.14159
   Received: undefined
```

失败时，检查：
1. 是否完成了 TODO
2. 语法是否正确
3. 逻辑是否符合要求

## 借助 AI 学习

### 何时使用 AI

| 场景 | 建议 |
|------|------|
| 概念不理解 | ✅ 让 AI 解释 |
| 练习题卡住 | ⚠️ 先自己想 5 分钟 |
| 综合挑战 | ✅ 可以让 AI 辅助 |
| 代码报错 | ✅ 让 AI 帮忙分析 |

### AI 协作要点

1. **描述清楚需求** - 越具体越好
2. **提供上下文** - 相关代码、错误信息
3. **审核生成的代码** - 不要盲目复制
4. **理解为什么** - 问 AI 解释原理

## 学习节奏建议

### 初学者
- 每天 1-2 小时
- 每章 2-3 天完成
- 不要跳过概念文档

### 有经验者
- 快速浏览概念
- 直接做练习和挑战
- 遇到不会的再回看概念

## 常见问题

### Q: 测试一直失败怎么办？
A: 
1. 检查语法错误
2. 仔细阅读测试期望
3. 查看概念文档
4. 问 AI 帮忙分析

### Q: 可以跳过章节吗？
A: 可以，但建议按顺序学习，因为后面的内容可能依赖前面的概念。

### Q: 挑战太难怎么办？
A: 
1. 可以借助 AI 辅助
2. 分解成小问题逐个解决
3. 查看提示（如果有）
4. 先跳过，学完后面再回来
