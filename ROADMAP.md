# ForgeStack 开发计划

## 项目定位

AI 时代的 JavaScript/TypeScript 学习系统，降低学习成本，提升学习效率。

---

## 技术方案演进

### 阶段一：MVP（当前）

**形态**：纯文件形式

```
学员下载项目 → IDE 打开 → 阅读 .md → 编辑 .js → pnpm test
```

**技术栈**：
- 运行环境：Node.js 18+
- 测试框架：Vitest
- 模块系统：ES Modules
- 文档格式：Markdown

**优点**：
- 零部署成本
- 离线可用
- 快速迭代内容

**目标**：
- [ ] 完成 10 个章节内容
- [ ] 验证学习模式有效性
- [ ] 收集用户反馈

---

### 阶段二：文档网站（待定）

**形态**：静态文档网站

```
Markdown → VitePress → 静态网站 → 在线阅读
练习仍需本地运行
```

**技术栈**：
- 文档框架：VitePress 或 Docusaurus
- 部署：Vercel / Netlify / GitHub Pages

**优点**：
- 阅读体验提升
- 可在线分享
- SEO 友好

**触发条件**：
- 内容基本完善
- 有分享传播需求

---

### 阶段三：交互式平台（远期）

**形态**：在线学习平台

```
浏览器内编辑 → 即时运行 → 反馈结果
```

**技术栈（参考）**：
- 前端：React / Vue + Monaco Editor
- 后端：Node.js + 沙箱执行
- 数据库：用户进度存储

**优点**：
- 无需本地环境
- 完整学习体验
- 进度追踪

**触发条件**：
- 有足够用户量
- 有开发资源投入
