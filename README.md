# ForgeStack - 全栈开发学习项目

欢迎来到 ForgeStack！这是一个专注于 JavaScript 和 TypeScript 全栈开发的学习项目仓库。

## 📚 学习计划

详细的学习计划请查看：**[JavaScript & TypeScript 学习计划](learning_plan.md)**

这是一个为期120天的项目实战学习计划，涵盖：
- 🎯 **4个阶段**: 从基础到商业项目
- 💻 **8+个项目**: 前端、后端、全栈完整覆盖
- 🛠️ **现代技术栈**: React + TypeScript + Node.js
- 🚀 **实战导向**: 80% 项目实践 + 20% 理论学习

## 🗂️ 项目结构

```
forgestack/
├── README.md                           # 项目介绍
├── JavaScript-TypeScript-学习计划.md    # 详细学习计划
├── projects/                           # 学习项目目录
│   ├── phase1-frontend/               # 第一阶段：前端项目
│   ├── phase2-backend/                # 第二阶段：后端项目
│   ├── phase3-fullstack/              # 第三阶段：全栈项目
│   └── phase4-commercial/             # 第四阶段：商业项目
├── notes/                             # 学习笔记
└── resources/                         # 学习资源
```

## 🚀 快速开始

1. **查看学习计划**: 阅读 [JavaScript & TypeScript 学习计划](learning_plan.md)
2. **选择起点**: 根据你的基础选择合适的阶段开始
3. **创建项目**: 在对应的 `phase` 目录下创建你的项目
4. **记录学习**: 在 `notes/` 目录下记录学习心得

## 🎯 学习目标

通过这个学习计划，希望你能够：
- 掌握现代 JavaScript 和 TypeScript 开发
- 熟练使用 React 生态系统
- 构建 Node.js 后端 API 服务
- 完成全栈应用的开发和部署
- 独立完成商业级项目并上线

## 📈 学习进度

- [ ] 第一阶段：JavaScript 基础与前端核心（第1-30天）
- [ ] 第二阶段：TypeScript 与后端基础（第31-60天）
- [ ] 第三阶段：全栈整合与高级功能（第61-90天）
- [ ] 第四阶段：商业项目实战与上线（第91-120天）
