# ForgeStack JS/TS

> AI 时代的 JavaScript/TypeScript 学习系统

## 🎯 学习理念

在 AI 已能自动开发网站的时代，人类仍然应该认真学习 JavaScript 和 TypeScript，但学习方式需要转变：

| 传统学习 | AI 时代学习 |
|----------|-------------|
| 记忆语法细节 | **理解概念和模式** |
| 从零手写代码 | **描述需求 → 审核 AI 代码** |
| 追求编码速度 | **追求判断力和决策力** |
| 被动阅读示例 | **主动探索和验证** |

**未来的竞争力** = 理解系统 + 指导 AI + 审核代码 + 做关键决策

## 🔄 三层学习模式

每个章节包含三个学习层次：

### Layer 1: 概念理解 (`concepts/`)
- 阅读概念文档，建立心智模型
- 重点理解"是什么"和"为什么"
- 时间：约 15-20 分钟/节

### Layer 2: 动手练习 (`exercises/`)
- 完成填空和编码练习
- 运行测试验证答案
- 时间：约 30 分钟/节

### Layer 3: 综合挑战 (`challenges/`)
- 综合应用所学知识
- 可借助 AI 辅助实现
- 重点：审核代码质量
- 时间：约 30-60 分钟/挑战

```bash
# ...
```

## 📝 学习建议

1. **不要跳过概念文档** - 理解比记忆更重要
2. **先自己思考再看答案** - 主动学习效果更好
3. **善用 AI 但要审核** - AI 是工具，你是决策者
4. **遇到问题先调试** - 调试能力是核心竞争力
