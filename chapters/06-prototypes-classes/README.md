# 第六章：原型与类

## 📖 本章概述

理解 JavaScript 的原型机制和 ES6 class 语法，掌握面向对象编程。

## 🎯 学习目标

完成本章后，你将能够：

- 理解原型和原型链的工作原理
- 熟练使用 ES6 class 语法
- 掌握类的继承和 super 关键字
- 使用静态方法、getter/setter 和私有字段

## 🔑 核心 ES6+ 特性

| 特性 | 说明 |
|------|------|
| class | 类声明语法 |
| extends | 继承 |
| super | 调用父类 |
| static | 静态方法/属性 |
| `#` | 私有字段 |
| get/set | 访问器属性 |
