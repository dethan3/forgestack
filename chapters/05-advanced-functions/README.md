# 第五章：高级函数

## 📖 本章概述

掌握 JavaScript 函数式编程的核心技术，包括高阶函数、柯里化、函数组合等。

## 🎯 学习目标

完成本章后，你将能够：

- 理解高阶函数的概念和应用
- 掌握函数柯里化和部分应用
- 学会函数组合和管道操作
- 实现记忆化优化性能
- 掌握防抖和节流技术

## 🔑 核心概念

| 概念 | 说明 |
|------|------|
| 高阶函数 | 接收或返回函数的函数 |
| 柯里化 | 将多参函数转为单参函数链 |
| 函数组合 | 组合多个函数成一个 |
| 记忆化 | 缓存函数结果 |
| 防抖/节流 | 控制函数执行频率 |
