# 第十章：DOM 与事件

## 📖 本章概述

学习 DOM 操作和事件处理，将 JavaScript 应用到实际的网页交互中。

## 🎯 学习目标

完成本章后，你将能够：

- 熟练使用 DOM 查询和操作方法
- 理解事件流（捕获、目标、冒泡）
- 掌握事件委托和事件处理
- 实现动态页面交互
- 完成综合交互项目
