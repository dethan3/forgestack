## 🚀 如何完成练习

1. 阅读对应的 `concepts/` 文档
2. 打开练习文件，阅读每个函数的注释要求
3. 在 `// TODO` 处编写代码
4. 运行测试验证

## 🧪 运行测试

```bash
# 运行第一章所有测试
pnpm test:01

# 监听模式（代码保存后自动重跑）
pnpm test:watch chapters/01-fundamentals
```

- 每个练习文件都有对应的 `.test.js` 测试文件
